#  Supply Chain Performance Dashboard | Power BI

##  Project Overview

This project presents an interactive **Supply Chain Performance Dashboard** built in **Power BI** to analyze sales performance, delivery operations, customer behavior, and product profitability.

The dashboard enables stakeholders to monitor key business metrics, identify operational bottlenecks, and support data-driven decision-making through interactive visualizations.

---

##  Business Objectives

- Monitor overall sales and profitability
- Analyze delivery performance and shipping efficiency
- Identify top-performing products and customer segments
- Evaluate department and market performance
- Provide interactive insights through dynamic filtering

---

## 🛠️ Tools & Technologies

##  Data Model

This project follows a **Star Schema** consisting of:

- Fact_Orders
- Dim_Customers
- Dim_Products
- Dim_Date

The model is optimized for efficient filtering and DAX calculations.

---

##  Key Business Insights

- Identified high-performing product categories and markets.
- Monitored delivery performance using on-time and late delivery metrics.
- Compared actual vs. scheduled shipping times.
- Analyzed customer purchasing behavior across different segments.
- Evaluated departmental profitability to identify growth opportunities.
